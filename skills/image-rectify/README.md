# image-rectify

Local MAAT skill for rectifying images into a cleaned-up, presentation-ready version.

## Purpose
Use this skill when the user wants to straighten, crop, align, or otherwise rectify an image for better visual presentation.

## Guidance
- Preserve the original subject and meaning.
- Improve composition, alignment, and readability.
- Avoid inventing new objects or text.
- If text is present, keep it legible and faithful.
- Prefer subtle corrections over heavy edits unless the user asks otherwise.

## Typical workflow
1. Inspect the input image.
2. Identify skew, perspective, crop, and clarity issues.
3. Apply the minimum correction needed.
4. Verify the result looks natural and accurate.

## Notes
This skill is a lightweight instruction file for local use inside MAAT workflows.
